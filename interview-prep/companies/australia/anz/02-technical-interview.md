## üìã Interview Process

### Stage 1: Online Application
- CV upload on ANZ careers portal
- Profile creation

### Stage 2: Assessments
1. **Psychometric Assessment** - Game-based + personality
2. **Aptitude Test** - Logical reasoning, problem-solving

### Stage 3: Pre-screening (15-20 min)
- Hiring manager call
- CV clarification
- Role expectations

### Stage 4: Technical Rounds

**Round 1: Coding & Algorithms**
- Data structures (arrays, trees, graphs)
- Algorithms (sorting, searching, DP)
- Problem-solving approach
- Code quality

**Round 2: Technical Deep Dive**
- Java core concepts (OOP, collections, multithreading)
- Spring Boot (REST, Data, Security)
- Database design (joins, views, optimization)
- System design (for senior roles)
- Past project discussions

**Topics:**
- Java 8+ features (Streams, Lambda, Optional)
- Microservices patterns
- Cloud architecture (AWS/Azure)
- Design patterns
- Concurrency and threading
- Exception handling
- Maven/Gradle

### Stage 5: Competency-Based Interview
- STAR method answers
- Past behavior examples
- Problem-solving scenarios
- Teamwork and collaboration
- Motivation and career goals
- Knowledge of ANZ

### Stage 6: HR Round
- Communication skills
- Cultural fit
- Salary discussion
- Joining details

### Stage 7: Pre-employment Checks
- Background verification
- Identity, employment, police checks
- Right to work verification
