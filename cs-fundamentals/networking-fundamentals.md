## ğŸ”Ÿ Network Debugging Tools

### Essential Commands

```bash
# DNS lookup
nslookup google.com
dig google.com

# Test connectivity
ping google.com                  # ICMP echo
traceroute google.com            # Path to destination

# Port checking
telnet redis.server.com 6379     # Check if port is open
nc -zv localhost 5432            # Netcat port scan

# HTTP debugging
curl -v https://api.github.com   # Verbose HTTP request
curl -I https://google.com       # Headers only (HEAD request)

# Network connections
netstat -tulpn                   # List listening ports
ss -tulpn                        # Modern netstat
lsof -i :8080                    # What's using port 8080?

# Packet capture
tcpdump -i eth0 port 443         # Capture HTTPS traffic
wireshark                        # GUI packet analysis
```
