# LeetCode Patterns Visualizer

An interactive and premium web application designed to help developers master the top algorithmic problem-solving patterns frequently asked in coding interviews. 

## Features
- **10 Essential Patterns**: Covers Sliding Window, Two Pointers, Fast & Slow Pointers, Merge Intervals, Cyclic Sort, LinkedList Reversal, Tree BFS, Tree DFS, Two Heaps, and Subsets.
- **Interactive Visualizations**: Step-by-step visualizations of arrays, pointers, and mathematical operations.
- **Live Code Highlighting**: Synchronized Java code snippets that highlight the exact line being executed during the visualization.
- **Premium Design System**: Modern, dark-themed UI with glassmorphism effects and smooth micro-animations.

## How to Run Locally
1. Navigate to the project directory:
   ```bash
   cd projects/leetcode-patterns-visualizer
   ```
2. Start a local server:
   ```bash
   npx serve .
   ```
3. Open the provided `localhost` link in your browser.

## Technologies Used
- HTML5 Canvas & DOM Manipulation
- Vanilla CSS3 (Custom Variables, Flexbox/Grid, Animations)
- Vanilla JavaScript
- FontAwesome Icons
- Highlight.js for code formatting
